# **Welcome to CMKL Finance System**

## **Prerequisite**

1. Node.js
2. Yarn

## **How to run this project**

1.  Start server

    ```console
    $ cd ./server
    $ yarn
    $ touch .env
    $ vim .env # You can see which variable to add from .sample.env
    $ yarn run dev
    ```

2.  Start client

    ```console
    $ cd ./client
    $ yarn
    $ yarn start
    ```
