# **Welcome to cmkl finance system**

## **Prerequisite**

1. Node.js
2. Yarn

## **How to run this project?**

1.  Start server

    ```console
    $ cd ./server
    $ yarn
    $ yarn run dev
    ```

2.  Start client

    ```console
    $ cd ./client
    $ yarn
    $ yarn start
    ```
